# 🧠 Text Annotation Demo (Emotion-Focused)

## 📌 Objective
This example demonstrates how text annotation can be used to identify **emotion, tone, and factual content** within a paragraph.  
All emotional expressions are highlighted.

---

## 📝 Annotated Paragraph

I expected the service to be much faster,  
but <mark>I'm honestly very disappointed with the experience</mark>.  
The support team responded late and didn’t provide a useful solution,  
which caused delays in completing my task.  
I hope this improves in the future.

---

## 🎯 Annotation Breakdown

- **<mark>Emotion (Highlighted)</mark>**:  
  - "I'm honestly very disappointed with the experience" → Negative emotion (disappointment)

- **Context**:  
  - "I expected the service to be much faster"

- **Tone (Critical)**:  
  - "The support team responded late and didn’t provide a useful solution"

- **Factual Impact**:  
  - "which caused delays in completing my task"

- **Intent / Suggestion**:  
  - "I hope this improves in the future"

---

- Emotion Detection  
- Text Annotation  
- Sentiment Analysis  
- Quality Evaluation
